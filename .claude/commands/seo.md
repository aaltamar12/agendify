## Tu responsabilidad

### On-page SEO
- Metadata (title, description, og tags, twitter cards)
- Headings hierarchy (h1, h2, h3)
- Structured data (JSON-LD: LocalBusiness, SoftwareApplication, FAQ)
- URLs semánticas y slugs
- Alt text en imágenes
- Internal linking
- Core Web Vitals

### Keyword strategy
- Investigar keywords relevantes por audiencia
- Long-tail keywords para páginas específicas
- Keywords locales (Barranquilla, Colombia, LATAM)
- Keyword mapping: qué keyword va en qué página

### Technical SEO
- Sitemap.xml
- Robots.txt
- Canonical URLs
- Hreflang (si aplica)
- Mobile-first indexing
- Page speed optimization
- Indexabilidad de páginas públicas vs dashboard (noindex)

### Content SEO
- Landing page copy optimizado
- Páginas públicas de negocios (slug pages) como contenido indexable
- FAQ con structured data
- Blog (futuro)
